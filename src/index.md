    <h1>From Sputnik to Starlink: A Data-Driven History of Satellites</h1>
    <p>
    The history of satellite launches spans nearly seven decades, beginning with the Soviet Union‚Äôs launch of Sputnik 1 in 1957, the first artificial satellite in human history. In the decades that followed, the United States and the USSR dominated orbital launches, driven primarily by military and scientific competition during the Cold War.
The dataset used here, via <a href="https://www.ucs.org/resources/satellite-database" target="_blank">Satellite Database | Union of Concerned Scientists</a>,  begins in 1974, well after the first recorded launch. The early entries are sparse‚Äîonly one satellite launch recorded in 1974, followed by a long gap until 1988‚Äîreflecting incomplete coverage, not a true absence of launches. We also notice incomplete coverage in the later years; however, once more continuous data begins, we can analyze the trends to uncover global technological, economic, and geopolitical developments. 
    </p>

  <hr class="divider">
    
  <h3> Slow and Steady Growth (1988 - Early 2000s)
</h3>
  <p>
   From the late 1980s into the mid-2000s, satellite launches increased gradually across the world. Improvements in electronic size reduction, lower launch costs, and growing commercial demand all contributed to this steady rise. The number of satellites launched per year slowly climbed from 1‚Äì3 launches per year in the late 1980s
to 17 by 1997
to around 20‚Äì30 launches per year by the early 2000s.

Why did this happen? A key driver of this period was the communications boom. Companies and governments began deploying satellites for television broadcasting and early mobile networks, among other use cases. According to Via Satellite‚Äôs (<a href="https://interactive.satellitetoday.com/via-satellite-at-30-the-1990s-and-early-2000s/" target="_blank">Via Satellite at 30: The 1990s and Early 2000s - Via Satellite</a>) retrospective analysis of the 1990s and early 2000s, demand especially surged in Asia, where rapidly developing economies invested heavily in telecommunications infrastructure. Some prolific operators from this period include: Eutelesat (Europe), Inmarsat (UK), AsiaSat (Hong Kong), IntelSat (more global).

  <h3 style="font-weight: 800;">U.S. vs. Rest of World Launches</h3>
  <p>
    For most years up to around 1990, the United States accounted for 100% of launches in the available data. But this changed in the mid-1990s and especially after the year 2000.
The percent of launches by ‚ÄúRest of World‚Äù (i.e. all countries except the US) climbed from 
0‚Äì30% in the early 1990s 
 to 50‚Äì65% in the early 2000s 
to 81% in 2014 
This shift reflects the rise of satellite programs in Europe, Russia, China, Japan, and India, among others, and the wider commercialization of space. The United States remained a major player, but by the late 1990s and early 2000s, many other countries were actively contributing to global satellite deployment.
This diversification of countries was driven by a few factors, including expanded commercial telecommunications, the growth of national, government-funded space programs, and lower launch costs as technologies advanced.

  </p>

  <h3> A Star-like Explosion (2017‚Äì2023) </h3>
  <p>
    After decades of gradual growth, the number of satellites launched suddenly skyrocketed:
2016: 107 satellites

2017: 221

2018: 308

2019: 272

2020: 1,054

2021: 1,590

2022: 2,113

2023: 909

This dramatic increase does not reflect more countries building satellites‚Äîit reflects a single tectonic shift in the industry:
The rise of mega-constellations, in particular SpaceX Starlink.
Beginning in 2020, SpaceX began launching 50‚Äì60 satellites per mission, sometimes completing multiple launches per month. This effectively rewrote the scale of global satellite activity. (<a href="https://spacenews.com/spacex-passes-2000-starlink-satellites-launched)" target="_blank">SpaceX passes 2,000 Starlink satellites launched - SpaceNews</a>)
To illustrate:
Starlink alone has launched over 5,000 satellites, more than every other country combined.

In 2022, the U.S. accounted for ~86% of all satellites launched (1664 )‚Äîalmost entirely because of Starlink.

The global satellite count doubled in less than three years.

Other companies also began building constellations (e.g., OneWeb, Amazon‚Äôs Kuiper, Planet), but none approached Starlink‚Äôs scale.
Because of Starlink, annual global launch numbers went from hundreds to thousands almost overnight. These numbers are not slowing down either, with one article stating that, ‚ÄúIndustry experts project that by the end of 2025, global satellite launches could exceed 3,000, marking an unprecedented annual milestone.‚Äù (<a href="https://orbitaltoday.com/2025/06/16/satellite-launches-soar-to-record-levels-in-2025-as-new-players-join-the-race/" target="_blank">Record-Breaking 1,200+ Satellites Launched in 2025</a>).

import { createSatelliteGlobe2 } from "./satplot.js";

const mapDiv = document.querySelector(".page .map");
const labelEl = mapDiv.querySelector(".map-label");

const globe = await createSatelliteGlobe2({
  container: mapDiv,
  FileAttachment,
});

const { satellites, data } = globe;

// Which color to use for each globe country in the orbit view
const orbitColors = new Map([
  ["United States of America", "#4000ffff"],
  ["China", "#ff0000ff"],
  ["Germany", "#00ff48ff"],
  ["Australia", "#23c7caff"],
  ["ALL", "#999999"],
]);

// Map from topojson / HTML country name -> operatorCountry values in the data
const operatorCountryAliases = new Map([
  ["United States of America", ["USA", "United States"]],
  ["China", ["China", "People's Republic of China"]],
  ["Germany", ["Germany"]],
  ["Australia", ["Australia"]],
]);

// map from HTML data-country -> topojson country name
const globeCountryAliases = new Map([
  ["USA", "United States of America"],
  // add others if you ever need them:
  // ["UK", "United Kingdom"],
]);

function toGlobeCountryName(name) {
  if (!name) return null;
  return globeCountryAliases.get(name) ?? name;
}
function satellitesForStep(dataset) {
  const allSats = satellites;
  let result = [];

  // 1. which countries‚Äô orbits to show?
  let topoNames = [];
  if (dataset.countries) {
    topoNames = dataset.countries
      .split(",")
      .map((s) => s.trim())
      .filter(Boolean);
  } else if (dataset.country) {
    topoNames = [dataset.country];
  }

  if (topoNames.length === 0) {
    // no country filter: start with everything
    result = allSats.slice();
  } else {
    // precompute USA operator codes for exclusion
    const usaOps = operatorCountryAliases.get("United States of America") ?? [
      "USA",
      "United States",
    ];

    for (const topoName of topoNames) {
      const color = orbitColors.get(topoName) ?? "#ff0000";

      if (topoName === "ALL") {
        // üîπ ALL = everyone EXCEPT USA
        const subset = allSats.filter(
          (d) => !usaOps.includes(d.operatorCountry)
        );
        subset.forEach((d) => {
          d.orbitColor = color;
        });
        result.push(...subset);
      } else {
        // normal per-country behavior
        const allowedOps = operatorCountryAliases.get(topoName) ?? [topoName];
        const subset = allSats.filter((d) =>
          allowedOps.includes(d.operatorCountry)
        );
        subset.forEach((d) => {
          d.orbitColor = color;
        });
        result.push(...subset);
      }
    }
  }

  // 2. time filter
  if (dataset.time) {
    const [startStr, endStr] = dataset.time.split("-");
    const startYear = startStr ? +startStr : -Infinity;
    const endYear = endStr ? +endStr : Infinity;

    result = result.filter(
      (d) =>
        d.launchYear != null &&
        d.launchYear >= startYear &&
        d.launchYear <= endYear
    );
  }

  // 3. purpose filter, if any
  if (dataset.purpose) {
    const wanted = dataset.purpose
      .split(",")
      .map((s) => s.trim().toLowerCase())
      .filter(Boolean);

    result = result.filter((d) =>
      wanted.includes((d.purpose || "").toLowerCase())
    );
  }

  return result;
}

/* all the step elements */ const steps = document.querySelectorAll(".step");
const text = document.querySelector(".context");

const interaction = new IntersectionObserver(
  (entries) => {
    entries.forEach((entry) => {
      if (!entry.isIntersecting) return;

      const ds = entry.target.dataset;
      const { lon, lat, scale, country } = ds;

      const showLaunches =
        ds.showLaunches === "true" ||
        ds.showLaunches === "on" ||
        ds.showLaunches === "1" ||
        ds.showLaunches === "ALL";

      if (lon && lat) {
        globe.flyTo({
          lon: +lon,
          lat: +lat,
          scale: scale ? +scale : undefined,
        });
      }

      if (country) {
        globe.highlightCountry(country);
        globe.highlightSatelliteCountry(country);

        const isAllLaunches = ds.showLaunches === "ALL";

        if (showLaunches) {
          let launchCountry;
          if (isAllLaunches) {
            launchCountry = "ALL";    // show all launch sites
          } else {
            launchCountry = country;  // show only that country's sites
          }
          globe.drawLaunchSites(launchCountry);
        } else {
          globe.drawLaunchSites(null);
        }
      } else {
        globe.highlightCountry(null);
        globe.highlightSatelliteCountry(null);
        globe.drawLaunchSites(null);
      }

      // satellites for this step
      const satsToShow = satellitesForStep(ds);
      globe.drawSatellites(satsToShow);

      // üîπ build label text
      const pieces = [];

      // country / countries (if you use data-countries, you can surface that too)
      if (ds.countries) {
        const countryList = ds.countries
          .split(",")
          .map((c) => c.trim())
          .filter(Boolean)
          .map((c) => {
            // map color code to a human-readable color name (optional)
            const colorHex = orbitColors.get(c) ?? "#000000";

            // Optionally convert hex to a color name
            const colorName = (() => {
              switch (colorHex.toLowerCase()) {
                case "#4000ffff":
                  return "red";
                case "#ff0000ff":
                  return "blue";
                case "#00ff48ff":
                  return "green";
                case "#23c7caff":
                  return "cyan";
                case "#999999":
                  return "gray";
                default:
                  return colorHex; // fallback to hex if no name
              }
            })();
            if (c == "ALL") {
              c = "Rest of World";
            }
            return `${c} (${colorName})`;
          });

        pieces.push(`Countries: ${countryList.join(", ")}`);
      } else if (country) {
        pieces.push(`Country: ${country}`);
      }

      // time range
      if (ds.time) {
        pieces.push(`Years: ${ds.time}`);
      }

      // satellite count
      pieces.push(`Satellites: ${satsToShow.length}`);

      // launch site count for the highlighted country (if any)
      if (country && showLaunches) {
        const relevantLaunches = data.filter(
          (d) => d.launch_country === country
        );
        const siteCount = new Set(
          relevantLaunches
            .map((d) => d["Launch Site"] && d["Launch Site"].trim())
            .filter(Boolean)
        ).size;

        pieces.push(`Launch sites: ${siteCount}`);
      }

      // update the label element
      labelEl.textContent = pieces.join(" ‚Ä¢ ") || "Global view";
    });
  },
  {
    root: text,
    threshold: 0.6,
  }
);

steps.forEach((s) => interaction.observe(s));
